# Contributing to VELORA

Thank you for your interest in contributing to VELORA! This project is a premium streaming interface built with modern web technologies.

## 🛠️ Development Setup

1. **Clone the Repo**: 
   ```bash
   git clone <repo-url>
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Environment Setup**:
   Copy `.env.example` to `.env` and provide your `VITE_TMDB_API_KEY`, `VITE_SUPABASE_URL`, and `VITE_SUPABASE_ANON_KEY`.

4. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## 🏗️ Technical Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + Lucide Icons
- **Backend**: Supabase (Auth/Database)
- **Data**: TMDB API

## 🧹 Code Standards

- **TypeScript**: Ensure all new code is type-safe.
- **Components**: Use functional components and hooks.
- **Styling**: Follow the existing design system (Glassmorphism, Red branding).
- **Linting**: Run `npm run lint` before committing.

## 🚀 Pull Requests

1. Create a branch for your feature/fix.
2. Ensure your changes are documented if they add new functionality.
3. Submit a PR with a clear description of the change.
